# Cursr

A custom mouse case with precise cutouts for buttons, the scroll wheel, and screws, featuring a custom shape made specifically for the Bambu Lab Wireless Mouse Components Kit.

## Purpose
I designed this project to learn how to use Onshape for CAD modeling, specifically focusing on how to build custom enclosures and cases for PCBs and other hardware devices.

## How to Use
1. Insert the Bambu Lab mouse PCB and a battery into the bottom shell.
2. Make sure the buttons and scroll wheel are properly aligned with the cutouts in the case.
3. Align the screw holes and tighten the screws to secure the board inside.
